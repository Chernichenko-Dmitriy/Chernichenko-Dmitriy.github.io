# Semantic HTML Resume Assignment

This project is a single-page personal resume website created as part of the assignment, “Build a Semantic HTML Resume with AI.” It was designed to demonstrate strong semantic HTML structure, accessible content organization, and modern CSS styling.

## Assignment Goal

The purpose of this assignment was to turn raw resume information into a professional, accessible, and well-structured resume webpage using:

- Semantic HTML elements such as header, nav, main, section, article, aside, and footer
- A clear heading hierarchy from h1 through h2 and beyond where appropriate
- Accessible lists and formatted dates using time elements
- Clean modern CSS with responsive layout techniques

## What This Project Includes

- A polished resume layout with sections for contact information, skills, certifications, education, projects, and experience
- A dark, modern visual theme with responsive design for desktop and mobile screens
- Accessibility support including a skip link and visible keyboard focus states
- Semantic content grouping to reflect professional resume structure

## Files

- index.html — Contains the semantic structure and resume content
- style.css — Contains the visual design, spacing, layout, and responsive styling
- README.md — Project overview and assignment context

## How to View the Project

1. Open the project folder in a browser.
2. Open index.html directly, or use a simple local server if preferred.
3. No installation or build tools are required.

## Assignment Requirements Covered

This page was built with the assignment requirements in mind:

- Header and footer are used for site identity and closing information
- Main content is organized into meaningful sections
- Experience entries are presented as distinct content blocks
- Dates are formatted with semantic time elements
- The design uses modern CSS rather than outdated layout methods
